# SOC Investigation #008

# README

## Investigation Title

Rogue Access Point Investigation

---

## Scenario

The Security Operations Center (SOC) received an alert from the organization's Wireless Intrusion Detection System (WIDS) indicating that an unauthorized wireless access point was detected within the corporate network.

The access point was broadcasting an SSID similar to the company's official wireless network, raising concerns about a potential Rogue Access Point or Evil Twin attack.

Your task is to investigate the alert, determine whether the device is authorized, assess the associated risks, and recommend the appropriate response.

---

## Investigation Objectives

- Validate the wireless security alert.
- Identify the rogue access point.
- Compare the rogue SSID with the legitimate corporate SSID.
- Review connected devices.
- Assess the security impact.
- Recommend containment and remediation actions.
- Document the investigation following SOC procedures.
