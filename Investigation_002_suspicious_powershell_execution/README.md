# Investigation 002 – Suspicious PowerShell Execution

## Scenario

A Security Information and Event Management (SIEM) system generated the following alert:

> **PowerShell executed a Base64-encoded command on a company workstation.**

PowerShell is a legitimate Windows administration tool, but attackers frequently abuse it to execute malicious scripts, download malware, and evade detection.

This investigation documents the steps taken to validate the alert, collect evidence, assess the severity, and recommend appropriate actions.

---

## Objectives

- Validate the alert
- Collect relevant evidence
- Assess the severity
- Recommend containment actions
- Document findings
