# Lessons Learned

## What I Learned

- PowerShell is a legitimate administrative tool but is commonly abused by attackers.
- Encoded PowerShell commands should always be investigated because they may indicate malicious activity.
- SIEM alerts provide the starting point for investigations but require supporting evidence before conclusions are made.
- Proper documentation ensures investigations are clear, consistent, and easy for other analysts to follow.
- Escalating suspicious activity is an important responsibility of a Tier 1 SOC Analyst.
