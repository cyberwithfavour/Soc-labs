# Investigation 004 – Suspicious DNS Traffic

## Scenario

A Security Information and Event Management (SIEM) solution generated the following alert:

> **One workstation generated over 500 DNS requests to an unknown domain within 10 minutes.**

Excessive DNS requests to an unfamiliar domain may indicate malware communication, DNS tunneling, or command-and-control (C2) activity.

This investigation documents the steps taken to validate the alert, collect evidence, assess the severity, and recommend appropriate actions.

---

## Objectives

- Validate the alert
- Analyze DNS activity
- Collect relevant evidence
- Assess severity
- Recommend containment actions
- Document findings
