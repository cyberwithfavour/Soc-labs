# Lessons Learned

## What I Learned

- DNS traffic should not be ignored during security investigations.
- A high volume of DNS requests may indicate malware communication or DNS tunneling.
- Understanding normal network behavior makes it easier to identify anomalies.
- Network traffic analysis complements endpoint investigations and provides additional context.
- Proper documentation helps ensure investigations are clear, consistent, and repeatable.
