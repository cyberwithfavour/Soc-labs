# Investigation 003 – Unauthorized Administrator Account Creation

## Scenario

A Security Information and Event Management (SIEM) solution generated the following alert:

> **A new local administrator account was created at 2:17 AM outside normal business hours.**

Creating administrator accounts outside approved maintenance windows is unusual and could indicate unauthorized access or privilege escalation.

This investigation documents the steps taken to validate the alert, collect evidence, assess the severity, and recommend appropriate actions.

---

## Objectives

- Validate the alert
- Identify relevant Windows Event IDs
- Collect supporting evidence
- Assess severity
- Recommend containment actions
- Document findings
