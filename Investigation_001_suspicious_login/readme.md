# Investigation 001 – Suspicious Login Investigation

## Scenario

A Security Information and Event Management (SIEM) solution generated an alert after detecting:

> **15 failed login attempts followed by one successful login from Germany.**

This investigation demonstrates the standard workflow a Tier 1 SOC Analyst would follow to validate, investigate, document, and escalate a suspicious authentication event.

---

## Objectives

- Validate the alert.
- Gather evidence.
- Assess severity.
- Recommend containment actions.
- Document findings.
