# 📚 Learning Methodology

For every investigation, I follow a structured SOC workflow:

1. Receive the alert
2. Validate the alert
3. Gather evidence
4. Analyze findings
5. Determine severity
6. Recommend actions
7. Document the investigation
8. Capture lessons learned
